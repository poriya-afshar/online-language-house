# Online Language House

This is a website project for an **online language school**, built solely for **introducing the school**, displaying courses, teachers, and conducting a placement test.

## Features

- Modern homepage introducing the language school
- Course catalog with different levels
- Teachers / instructors section
- Online **placement test** (level determination)
- Contact form
- Full content management via Django Admin panel
- Fully responsive design using pure CSS (no front-end framework)

## Technologies Used

### Back-end

- **Python 3**
- **Django 6** (core framework)
- SQLite (database)

### Front-end

- **HTML5**
- **CSS3** (no Bootstrap, Tailwind, or any CSS framework)
- **Pure JavaScript** (no React, Vue, or other JS frameworks)

### Other Tools

- Font Awesome for icons
- Google Fonts (Playfair Display, DM Sans, Vazirmatn)
